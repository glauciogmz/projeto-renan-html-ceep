Site Institucional - CEEP

Sobre o Projeto:

Este projeto consiste no desenvolvimento de um site institucional voltado ao CEEP (Centro Estadual de Educação Profissional Professora Lourdinha Guerra), com o objetivo de apresentar informações relevantes sobre a instituição de forma clara, organizada e acessível.

O site foi construído utilizando HTML e CSS, seguindo princípios fundamentais do desenvolvimento web, como separação de responsabilidades, organização de arquivos e uso de estrutura semântica.

Além disso, o projeto foi desenvolvido em equipe, utilizando o GitHub como ferramenta de versionamento e colaboração, permitindo que todos os integrantes contribuíssem de forma integrada para a construção do sistema.

O resultado final é um site com múltiplas páginas interligadas, navegação funcional, design responsivo e elementos interativos que simulam um ambiente real de aplicação web.

Objetivos do Projeto

Objetivo geral:
Desenvolver um site institucional completo que represente o CEEP, aplicando na prática os conhecimentos adquiridos em HTML e CSS.

Objetivos específicos:

- Apresentar a instituição e sua importância educacional  
- Desenvolver múltiplas páginas conectadas por navegação  
- Aplicar estrutura semântica HTML (main, footer)  
- Utilizar corretamente tags obrigatórias exigidas no projeto  
- Criar formulários com diferentes tipos de entrada de dados  
- Implementar responsividade para diferentes dispositivos  
- Trabalhar colaboração em equipe com GitHub  
- Organizar código de forma limpa e reutilizável  

Como Executar o Projeto

O projeto está disponível no GitHub e pode ser acessado assim:

1. Acesse o repositório no GitHub  
2. Clique no botão "Code"  
3. Escolha uma das opções:
   - Download ZIP
   - Ou copiar o link para clonar com Git  
4. Extraia os arquivos (se necessário)  
5. Abra a pasta do projeto  
6. Execute o arquivo   

O site será aberto automaticamente no navegador padrão do sistema.
